
# boilerplate-typescript-react
 Boilerplate for react in typescript

### Features
- Webpack3
- Typescript
- React
- SASS
- Hot Module Replacement (HMR)
- Tree Shaking
- React CSS Themr
- Storybook
- TSLint
- TS Node

#### Other features
- .nvmrc
- .editorconfig
- .gitignore
- .npmignore
- Modify SASS variables of external library
- Modify internal component style from parent (react-css-themr)
